Understanding the <iframe> Tag

The <iframe> tag is used to embed another webpage, video, map, or external content inside your webpage. It stands for inline frame.

Example 1: Embedding a Website

This example shows how to embed a webpage inside another webpage:

<iframe src="https://www.example.com" title="Example Website"></iframe>

Example 2: Embedding a YouTube Video

YouTube provides a special embed URL using iframes:

<iframe 
    src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
    title="YouTube Video"
    allowfullscreen>
</iframe>

Useful Attributes

  • src: the link to the page or content you want to display
  • width / height: control the iframe size
  • title: adds accessibility information
  • allowfullscreen: allows full screen mode for videos
  • loading="lazy": delays loading until the frame is visible
